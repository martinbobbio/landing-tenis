# Descripcion
Breve página web en la cual se muestra información acerca de *rankings*, *resultados*, *posteos* entre otros. Tambien los usuarios cuentan con un login para poder ver su perfil en la página.

Tecnologias usadas:
- Boostrap
- Ajax
- Jquery
